# SQL PROJECT WORLD LAYOFFS
This project consists of a detailed data cleaning and Exploratory Data Analysis (EDA) on a global layoffs dataset (layoffs_staging2). 
The primary objective is to identify patterns, temporal trends, and the most affected sectors during the 2020-2023 period.

By leveraging advanced SQL techniques such as CTEs and Window Functions, we explore the impact by industry, the rolling total of layoffs, and an annual ranking of companies with the highest workforce reductions. This analysis provides a clear perspective on the scale of the tech labor crisis in the post-pandemic era.
